# Contributing back to the core

Built something good in your `local/` that others should have? Offer it up:

1. Ask Claude to "send this skill to the core." It will package the file and
   open a pull request (you do not need to know git).
2. An automated scan checks for secrets and anything sketchy.
3. The maintainer reviews it and, if it is good, merges it into core.
4. On merge it reaches everyone on their next update.

The maintainer's review is the quality gate. Nothing reaches core without it.
